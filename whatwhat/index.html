Whatwhat

whatwhat is a command-line tool used for project tracking and management in the Research Engineering group at The Alan Turing Institute.

Overview

The reporting page contains a more thorough discussion of what whatwhat aims to accomplish, along with complete explanation of the error codes and how to fix them.

  • Whatwhat.Domain Domain model

Obtaining high-level overviews

  • Whatwhat.Project Pretty-print a project.
  • Whatwhat.Person Pretty-print a person.

Fetching data from Forecast

  • Whatwhat.ForecastRaw Raw interface to Forecast. Returns typed data that is essentially the JSON form returned by Forecast.
  • Whatwhat.Forecast High-level interface to the Forecast API.

Fetching data from GitHub

  • Whatwhat.GithubRaw GithubRaw queries the GitHub GraphQL and REST APIs for data about issues, users, and project boards, and parses that into OCaml records.
  • Whatwhat.Github High-level interface to the Project Tracker in Github.

Merging Forecast and GitHub data

  • Whatwhat.Schedule Schedule collates data from both GitHub and Forecast, and joins them into a single, source-agnostic data structure. It deals with people, projects, assignments, and allocations.

Output

  • Whatwhat.ForecastExport Functionality for exporting Forecast CSVs.
  • Whatwhat.Log Maintains a log of problems found with each project.
  • Whatwhat.QueryReports
  • Whatwhat.Notify Notify is used to tell people about problems
  • Whatwhat.Slack
